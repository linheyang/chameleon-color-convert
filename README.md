# chemeleon-color-convert
方便快捷的色彩空间转换器



## 预览

![Preview](https://github.com/linheyang/chemeleon-color-convert/blob/master/screenshots/mainprogram.gif?raw=true)

## 功能

* RGB、CMYK、HEX、HSB四种颜色模式进行转换

* 根据色值匹配最相近的CSS颜色名（共147种颜色名）

* 背景色为当前颜色预览

* 可置顶
